Simple Selectors(h1)

Subheading(h2)

The selectors are used to determine which HTML element or set of elements to apply CSS declarations. Element Selector,class selector and id selector are the various types of selectors.

Subheadings(h2)

Element selector is done by specifying the element name. Class Selector is specified with a dot and name of the class. Id selector is specified using a pound(#) sign in front of the id value.

id="mainpoint"
